## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+)
- Yarn or npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rishabhjaiswal3/my-portfolio.git
   ```

2. Install dependencies:
   ```bash
   yarn install
   # or
   npm install
   ```

3. Start the development server:
   ```bash
   yarn dev
   # or
   npm run dev
   ```

4. Build for production:
   ```bash
   yarn build
   # or
   npm run build
   ```
